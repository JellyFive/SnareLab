# 跟练页 UI 功能说明书

## 页面定位

跟练页是实际练习发生的页面。

它回答：

> 我现在怎么练？速度多少？打到哪里了？

## 核心功能

| 功能 | 说明 |
|---|---|
| 五线谱 / 节奏谱显示 | 用户直接看谱练习 |
| 播放控制 | 开始、暂停、继续、重置 |
| 播放线 | 红色播放线标识当前位置 |
| 当前拍点 | 顶部圆点显示当前拍 |
| BPM 调节 | 拨盘、+1、-1、+5、-5、自动升速 |
| 声音层开关 | 节拍器、鼓点、重音、无尽 |
| 长按编辑 | 长按音符局部编辑 |
| 保存记录 | 练习结束后保存练习记录 |

## 页面展示内容

### 谱面区域

- 五线谱 / 节奏谱
- 小节线
- 音符
- 休止符
- 重音高亮
- 红色播放线
- 当前拍点圆点

### 播放控制区

- 开始播放
- 暂停
- 重置
- 保存练习记录

### BPM 区

- BPM 拨盘
- 当前 BPM 大数字
- -1 / +1
- -5 / +5
- 自动升速

### 声音层开关

- 无限
- 重音
- 节拍器
- 鼓点

## 关键交互

| 交互 | 结果 |
|---|---|
| 点击开始播放 | 进入 count-in 后开始播放 |
| 点击暂停 | 暂停播放线和声音 |
| 点击重置 | 回到第一小节第一拍 |
| 调整 BPM 拨盘 | 改变当前练习速度 |
| 点击 +1 / -1 | 微调 BPM |
| 点击自动升速 | 开启 tempo ramp |
| 长按音符 | 打开局部编辑浮层 |
| 点击保存练习记录 | 打开练习回顾弹窗 |

## 长按编辑浮层

支持操作：

- 改为休止
- 添加 / 删除重音
- 设置 R / L
- 标记 ghost note
- 删除音符
- 拆分节奏，P2

## 空状态

| 场景 | 显示 |
|---|---|
| 没有练习数据 | 提示“请先从节奏页生成练习” |
| 浏览器禁止音频 | 提示“点击开始后浏览器才允许播放声音” |

## 设计注意

- 谱面区域是主视觉中心。
- BPM 是跟练页主控，不放在节奏生成器中作为核心。
- 播放线必须明显，使用红色。

# UI 控件规范 V1.0

## 1. 设计基调

产品统一使用“深色练习驾驶舱”风格。

| 类型 | 规范 |
|---|---|
| 背景 | 深色冷调，主背景 `#080B10` |
| 模块 | 卡片式，圆角 20px，细描边 |
| 强调色 | 黄色 `#FFB020`，用于主操作、当前状态、关键数据 |
| 危险/播放线 | 红色 `#FF5A7A` |
| 完成/积极反馈 | 绿色 `#7CFFCB` |
| 信息文本 | 灰蓝 `#95A3B8` |
| 字体 | Inter / 系统字体 / 苹方 / 微软雅黑 |
| 整体气质 | 专注、稳定、工具感、练习驾驶舱 |

## 5. 按钮规范

### 5.1 按钮类型

| 类型 | 样式 | 用途 |
|---|---|---|
| Primary | 黄色背景，深色文字 | 主操作，如开始、生成、保存 |
| Secondary | 深蓝灰背景，浅色文字 | 常规操作 |
| Blue | 深蓝背景 | 播放、跟练类操作 |
| Ghost | 透明背景 | 低优先级操作 |
| Disabled | 40% 透明度 | 不可用状态 |

### 5.2 状态

| 状态 | 表现 |
|---|---|
| Default | 正常描边与背景 |
| Hover | 描边变黄色，文字变黄色；Primary 保持深色文字 |
| Active | 轻微下移 1px |
| Disabled | 透明度 0.4，禁止点击 |
| Loading | 后续可加入 spinner，按钮文案变为“处理中” |

### 5.3 尺寸

| 尺寸 | 高度 | 用途 |
|---|---|---|
| Small | 32px | 小标签操作 |
| Default | 44px 左右 | 普通按钮 |
| Large | 54px | 首页主入口、播放按钮 |
| Full | 宽度 100% | 侧栏或跟练主按钮 |

## 6. 卡片规范

### 6.1 基础卡片

- 背景：`linear-gradient(180deg, #111722, #0F1622)`
- 描边：1px `#28364A`
- 圆角：20px
- 内边距：20px
- 阴影：`0 12px 32px rgba(0,0,0,.18)`

### 6.2 卡片层级

| 层级 | 用途 |
|---|---|
| 主卡片 | 页面核心模块 |
| 次级卡片 | 列表项、快速入口 |
| 高亮卡片 | 当前练习、选中节奏型 |
| 信息卡片 | 进度、统计、Focus |

### 6.3 卡片内容结构

推荐结构：

```text
Card
├── Title
├── Description / Meta
├── Main content
└── Action row
```

## 7. 图标风格

### 7.1 图标原则

- 使用线性图标，不用拟物风格。
- 线宽建议 1.75px 到 2px。
- 圆角端点。
- 图标颜色默认 `--muted`。
- 激活状态使用 `--accent`。
- 危险状态使用 `--red`。
- 图标不单独承担语义，重要操作必须配文字。

### 7.2 推荐图标集合

后续开发建议使用：

- Lucide Icons
- Phosphor Icons
- Remix Icon

## 8. 表单控件

### 8.1 输入框

- 背景：`#0D131D`
- 描边：`#34455F`
- 圆角：14px
- 内边距：12px 14px
- 聚焦态：描边 `--accent`
- 错误态：描边 `--red`

### 8.2 Select

同输入框，用于类型、模式、标签筛选。

### 8.3 Textarea

- 最小高度 120px
- 用于练习笔记、问题记录、练习回顾

---

## 9. Tabs / Segment

### 9.1 Tab

用于分类筛选：

- 十六分音符
- 三连音
- 我的收藏
- 全部 / 预设 / 生成 / 收藏

激活态：黄色背景，深色文字。

### 9.2 Segment

用于互斥模式：

- 随机 / 顺序 / 无尽
- 2/4 / 3/4 / 4/4

激活态同 Primary。

---

## 10. Switch

用于声音层和模式开关：

| 状态 | 样式 |
|---|---|
| Off | 深色背景，灰色圆点 |
| On | 蓝灰背景，黄色圆点 |

常用开关：

- 节拍器
- 鼓点
- 重音
- Count-in
- 无尽模式

---

## 11. 音符卡片

### 11.1 用途

用于节奏型选择，是产品的核心视觉记忆点。

### 11.2 结构

```text
PatternCard
├── 节奏名称
├── 击打数量
├── 音符可视化
└── 圆点辅助
```

### 11.3 状态

| 状态 | 样式 |
|---|---|
| Default | 深色卡片，灰蓝描边 |
| Selected | 黄色描边，暗黄色渐变背景 |
| Disabled / Placeholder | 透明度 0.42，虚线描边 |
| Hover | 描边变黄色 |
| Favorite | 右上角星标，后续添加 |

---

## 12. 五线谱 / 节奏谱区域

### 12.1 谱面区域

- 背景：`#0C111A`
- 描边：`#34455F`
- 圆角：20px
- 最小高度：370px

### 12.2 播放线

- 颜色：`--red`
- 宽度：4px
- 带发光阴影

### 12.3 当前拍点

- 当前拍点圆点使用黄色填充
- 非当前拍点为空心灰白描边

---

## 13. 响应式规则

| 断点 | 行为 |
|---|---|
| 大于 1180px | 完整侧边栏 + 多栏布局 |
| 900px 到 1180px | 侧边栏收缩为图标栏 |
| 小于 900px | 隐藏侧边栏，显示底部导航 |
| 小于 560px | 卡片单列，按钮全宽，谱面压缩 |

原则：只改变布局，不改变视觉风格。

# 练习库 UI 功能说明书

## 页面定位

练习库用于管理所有可复用练习材料。

它回答：

> 我保存过哪些练习？我要从哪里继续？

## 核心功能

| 功能 | 说明 |
|---|---|
| 查看练习列表 | 展示全部保存练习 |
| 筛选 | 按预设、生成、自定义、收藏、目标筛选 |
| 查看详情 | 查看练习来源、模式、拍号、BPM、标签 |
| 开始跟练 | 从练习库直接进入跟练 |
| 复制编辑 | 基于现有练习创建副本 |
| 加入个人目标 | 将练习关联到目标 |
| 导入导出 | JSON 数据导入导出，P1 |

## 页面展示内容

### 顶部筛选

- 全部
- 预设
- 生成
- 自定义
- 收藏
- 目标

### 操作区

- 新建练习
- 导入 JSON
- 导出

### 练习列表

每项展示：

- 练习名称
- 来源
- 类型
- 默认 BPM
- 标签
- 是否关联个人目标

### 练习详情

展示：

- 来源
- 模式
- 拍号
- 默认 BPM
- 目标
- 标签

操作：

- 开始跟练
- 复制编辑
- 加入目标

## 关键交互

| 交互 | 结果 |
|---|---|
| 点击练习项 | 右侧详情更新 |
| 点击开始跟练 | 跳转跟练页 |
| 点击复制编辑 | 创建副本并进入节奏/编辑页 |
| 点击加入目标 | 打开目标选择弹窗 |
| 点击导出 | 导出 JSON |
| 点击导入 | 选择本地 JSON 文件 |

## 空状态

| 场景 | 显示 |
|---|---|
| 无练习 | 提示“从节奏页生成第一个练习” |
| 无收藏 | 提示“收藏练习后会显示在这里” |

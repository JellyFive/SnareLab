# 节奏页 UI 功能说明书

## 页面定位

节奏页负责选择节奏型，并组织成一次练习。

它回答：

> 我要用哪些节奏型练？按什么方式练？

## 核心功能

| 功能 | 说明 |
|---|---|
| 节奏型选择 | 使用音符卡片选择基础节奏型 |
| 分类筛选 | 十六分音符、三连音、我的收藏 |
| 多选 | 支持同时选择多个节奏型 |
| 练习模式 | 随机、顺序、无尽 |
| 练习队列 | 展示已选节奏型的练习顺序 |
| 生成练习 | 根据队列生成可跟练谱面 |
| 保存队列 | 保存为练习或个人目标 |

## 页面展示内容

### 左侧设置区

- 练习模式：随机 / 顺序 / 无尽
- 节奏设置：
  - 生成练习
  - 保存队列
- 拍号：
  - 2/4
  - 3/4
  - 4/4

### 节奏型选择区

每张音符卡片展示：

- 节奏名称
- 击打数量，如 `2/4`
- 音符可视化
- 圆点辅助
- 选中状态

分类 Tab：

- 十六分音符
- 三连音
- 我的收藏

### 练习队列区

展示：

- 已选节奏型
- 顺序编号
- 音符简写
- 圆点辅助

模式 Tab：

- 随机打乱
- 按顺序
- 每轮重洗

## 关键交互

| 交互 | 结果 |
|---|---|
| 点击节奏型卡片 | 选中 / 取消选中 |
| 点击随机 | 队列播放顺序随机 |
| 点击顺序 | 保持当前队列顺序 |
| 点击每轮重洗 | 每轮重新随机 |
| 点击生成练习 | 生成谱面并可进入跟练 |
| 点击保存队列 | 保存为练习库项目 |
| 拖拽队列项 | 调整顺序，P1 |

## 空状态

| 场景 | 显示 |
|---|---|
| 未选择节奏型 | 提示“请选择至少 1 个节奏型” |
| 我的收藏为空 | 提示“收藏常用节奏型后会显示在这里” |

## 设计注意

- 用户优先看到音符，不要只显示 `X-X-`。
- 圆点是辅助信息，不应喧宾夺主。
- 选中状态必须明显，使用黄色描边和暗黄色背景。

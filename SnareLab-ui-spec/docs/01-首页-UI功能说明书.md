# 首页 UI 功能说明书

## 页面定位

首页是个人练习驾驶舱，用于回答：

> 今天该练什么？

首页不负责完整编辑，也不负责详细统计，而是把个人目标、最近练习、当前 Focus 和快速入口聚合到一起。

## 核心功能

| 功能 | 说明 |
|---|---|
| 继续上次练习 | 展示上次练习名称、BPM、时长、目标重复次数 |
| 快速入口 | 随机读谱、顺序目标、新建练习笔记 |
| 当前个人目标 | 展示正在推进的目标 |
| 最近练习 | 展示最近练习记录 |
| 本周进度 | 练习时长、目标完成度、稳定 BPM |
| 当前 Focus | 自动聚合重点标签 |
| 下一步建议 | 根据目标和记录提示下一步 |

## 页面展示内容

### 顶部状态区

- 页面标题：`首页 · 个人练习驾驶舱`
- 描述：把个人目标、练习笔记、练习计划、最近记录和快速入口集中在一个页面
- 状态 Badge：
  - 今日练习时长
  - 连续练习天数

### 继续上次练习卡片

展示：

- 练习名称
- 练习类型：个人目标 / 随机读谱 / 自定义练习
- 上次 BPM
- 上次练习时长
- 目标重复次数
- 当前建议 BPM 圆形仪表

操作：

- 开始跟练
- 调整节奏
- 写练习笔记

### 快速入口

三个快捷卡片：

1. 随机读谱
2. 顺序目标
3. 新建练习笔记

### 当前个人目标

列表展示：

- 目标标题
- 目标 BPM
- 状态
- 是否可转为顺序练习

### 最近练习

列表展示：

- 日期
- 练习名称
- BPM
- 时长
- 声音配置
- 主观难度

### 本周进度

进度条：

- 练习时长
- 目标完成度
- 稳定 BPM

### 当前 Focus

标签展示：

- 十六分
- 重音
- Paradiddle
- 顺序练习
- 读谱反应

### 下一步建议

展示 1 到 3 条建议，每条包括：

- 建议标题
- 建议说明
- 可跳转操作

## 关键交互

| 交互 | 结果 |
|---|---|
| 点击“开始跟练” | 跳转跟练页 |
| 点击“调整节奏” | 跳转节奏页 |
| 点击“写练习笔记” | 跳转日志页并打开新建笔记 |
| 点击当前目标 | 打开目标详情 |
| 点击最近练习 | 查看练习记录详情 |

## 空状态

| 场景 | 显示 |
|---|---|
| 没有练习记录 | 显示“创建第一个练习”入口 |
| 没有个人目标 | 显示“新建个人目标”入口 |
| 没有 Focus | 使用最近选择的节奏型生成默认 Focus |

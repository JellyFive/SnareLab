# SnareLab Practice Log v0.3 产品需求文档

**版本：** v0.3  
**状态：** 已确认  
**产品：** SnareLab Practice Log  
**平台：** 本地优先 Web App / PWA

## 1. 产品定位

SnareLab 是面向鼓手的本地练习记录工具。V0.3 的目标不是增加训练参数，而是把“开始练习、完成归类、回看记录、理解趋势”做成低摩擦的日常闭环。

## 2. V0.3 目标

- 用户在今日页一眼看到今日练习时长和历史总练习时长。
- 用户可以快速开始、暂停、继续、停止和重置一次练习。
- 练习结束后，用户可选择分类、标签、笔记和图片完成记录。
- 用户可以按时间线回看记录，并通过分类或标签筛选。
- 用户可以在记录详情 Bottom Sheet 内修改记录的元数据和图片。
- 用户可以按月或年查看热力图、分类统计和标签统计。

## 3. 信息架构

主导航固定为：

```text
今日 / 记录 / 统计
```

全局设置入口位于三个主页面右上角。设置页当前仅实现：

- 分类管理
- 标签管理

“页面主题”作为预留项目展示“即将开放”，V0.3 不提供主题切换。

## 4. 页面需求

### 4.1 今日

页面标题为“Today”。内容顺序：

1. 今日练习时长与总练习时长。
2. 全宽主操作“开始练习”。
3. 当前月紧凑热力图及本月练习天数、累计时长摘要。
4. 今日练习记录列表。

规则：

- 今日记录只作快速回顾，不可从今日页进入编辑。
- Today 页标题和右侧日期使用英文（`Today`、`Jul 12`）；其余页面可见文案保持中文。
- 今日页直接向下完整展示当天所有练习记录，不提供“查看全部记录”展开入口。
- 当天无记录时显示“今天还没有练习记录”，并保留开始练习按钮。
- 存在已结束未保存草稿时，显示“继续保存”和“放弃”操作；放弃需要确认。

### 4.2 计时

计时页为独立沉浸式页面，包含返回、圆形计时器、状态文字与控制按钮。

状态：

```text
待开始 -> 进行中 <-> 已暂停 -> 已结束 -> 保存面板
```

规则：

- 支持开始、暂停、继续、停止和重置。
- 暂停时长不计入练习时长。
- 重置清空当前未结束计时，并需要确认。
- 运行中的计时不跨刷新恢复。
- 停止后进入保存 Bottom Sheet。

### 4.3 保存 Bottom Sheet

保存面板将一次计时转换为练习记录。

- 时长：只读。
- 分类：必选单选；提供“新增分类”。
- 标签：可选多选；提供“新增标签”。
- 笔记：可选多行文本。
- 图片：可选，最多 6 张；每张本地压缩后不超过 2MB。
- 保存记录：未选择分类时禁用。
- 稍后保存：保存为单条本地草稿。
- 放弃：需要确认。

新增分类或标签使用轻量子面板完成，创建成功后自动选中，不离开保存流程。

### 4.4 记录

记录页使用按日期分组的紧凑信息型时间轴，而非月历优先视图。

- 默认按日期倒序显示。
- 每个日期显示星期、日期节点；其下每条记录显示开始时间节点、分类、时长、标签摘要、笔记摘要和图片数量。
- 时间轴用于快速浏览日期和时间顺序，正文采用小字号与紧凑间距，不承担详情阅读。
- 筛选仅支持分类和标签；不提供日期范围、时长范围或全文搜索。
- 点击记录打开记录详情 Bottom Sheet。

### 4.5 记录详情 Bottom Sheet

详情和编辑继续使用 Bottom Sheet，不创建独立详情页。

查看模式显示：时长、开始时间、结束时间、分类、标签、笔记和图片网格。

编辑模式允许修改：分类、标签、笔记、图片。

编辑模式禁止修改：开始时间、结束时间、练习时长。

删除为硬删除，需确认并明确说明不可恢复。

### 4.6 统计

统计页顶部显示：

- 今日时长
- 连续练习天数
- 累计练习时长

统计页使用“概览 / 分类 / 标签”三个顶级页签。概览默认展示年度练习情况：年度摘要、全年热力图和可点击的月份总结列表。点击月份进入月度详情；月度详情展示完整自然月热力图、当月汇总、分类分布和近期练习记录；点击日期进入日练习详情。

日练习详情只使用现有练习数据：当天时长、练习次数、分类分布、记录时间、分类、标签、笔记和图片。V0.3 不新增 BPM、卡路里、心情、目标达成率或其他训练字段。

分类页展示年度分类时长、按月趋势和分类明细。标签页展示按练习时长统计的标签排行与标签组合分析。分类和标签的时长百分比均以当前统计范围内的总练习时长为分母；一条带多个标签的记录会将其完整时长分别计入每个关联标签，因此标签百分比可以合计超过 100%。

## 5. 分类、标签与设置

- 移除独立“分类与标签”主导航。
- 分类和标签管理迁移到设置页；保存、编辑、筛选面板保留快捷新增或选择能力。
- 系统分类“未分类”始终存在且不可删除。
- 删除分类时，其记录自动迁移至“未分类”。
- 删除标签时，仅移除其与记录的关联，练习记录保留。

## 6. 数据与离线规则

- 产品保持本地优先，无账户、无云同步、无服务端。
- 记录、分类、标签和图片附件保存到本地 IndexedDB。
- 图片以压缩后的 Blob 保存；单条记录最多 6 张，每张不超过 2MB。
- 已结束未保存记录可恢复；运行中计时不可恢复。
- 删除记录为硬删除，不提供回收站。

## 7. V0.2 到 V0.3 的变化

| 项目 | V0.2 | V0.3 |
| --- | --- | --- |
| 主导航 | 今日、日志、分类、统计 | 今日、记录、统计 |
| 今日页 | 今日汇总与列表 | 增加总练习时长和月热力图 |
| 日志结构 | 月历优先 | 时间线优先 |
| 筛选 | 分类、标签、日期、时长 | 仅分类、标签 |
| 详情 | Bottom Sheet | 保留 Bottom Sheet，并增加图片 |
| 分类管理 | 独立主页面 | 右上角设置页与保存/编辑快捷入口 |
| 统计 | 分类分布和热力图 | 年度概览下钻、分类页、标签页与时长统计 |
| 图片 | 不支持 | 本地多图附件 |

## 8. 非目标

V0.3 不包含：

- BPM、拍号、节拍器、节奏编辑器或训练目标
- 账户、云同步、社交分享、AI 分析
- 音频或视频附件
- 主题切换的实际功能
- 编辑开始时间、结束时间和练习时长

## 9. 验收标准

- 用户可在三步内从今日页开始练习并进入计时。
- 停止计时后，未选择分类不能保存，选择分类后可保存记录。
- 一条记录可保存、展示、编辑和删除最多 6 张本地图片。
- 记录页可按分类和标签筛选，点击记录打开 Bottom Sheet。
- 月/年统计的热力图、分类图和标签图随周期同步更新。
- 页面上的可见文案均为中文，`SnareLab` 品牌标识除外。

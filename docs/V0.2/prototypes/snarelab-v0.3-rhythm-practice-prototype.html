UI 规范总览

色彩规范

V0.3 新增字段

• BPM：保存练习时记录，可用于轻量统计。
• 拍号：默认 4/4，可在保存页调整。
• 练习目标：短文本，帮助复盘练习意图。
• 节奏备注：记录手法、弱点、下次调整。

交互原则

• 计时前不强制填写，保存时补充语义。
• 记录详情仍用 Bottom Sheet。
• Log 日历和统计热力图复用同一组件。
• 高阶信息轻量出现，不压过计时闭环。

图标风格

线性图标、2px stroke、44px 触控热区、主操作只保留一个高强调按钮。
今天→专注计时→保存节奏信息→日志复盘→统计节奏趋势
产品方向V0.3 进入 Rhythm Practice，但仍保持本地优先、无账号、无云同步、无 AI 分析。
视觉方向保留 V0.2 的清爽白底和蓝紫主色，用橙色只标记 BPM / 节奏信息，避免变成复杂训练仪表盘。
信息架构Today / Log / Category / Statistics 四导航不变，高阶字段通过保存页、详情页和统计补充进入。

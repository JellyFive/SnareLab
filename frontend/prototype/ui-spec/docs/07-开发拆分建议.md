## 建议开发顺序

1. 建立设计 Token 与基础组件。
2. 开发响应式布局外壳。
3. 开发首页。
4. 开发节奏型选择页。
5. 开发练习队列。
6. 开发跟练页和音频播放。
7. 开发练习库。
8. 开发日志与个人目标。
9. 开发设置与数据导入导出。
10. 后续接入 MIDI 和套鼓扩展。

本文件夹将原先的一份长页面原型拆分为独立页面：

| 文件 | 页面 |
|---|---|
| `index.html` | 首页 |
| `rhythm.html` | 节奏型选择与练习编排 |
| `trainer.html` | 跟练页 |
| `library.html` | 练习库 |
| `log.html` | 日志 |
| `settings.html` | 设置 |
| `style.css` | 全局样式与响应式规则 |

设计风格保持：深色练习驾驶舱、卡片式模块、冷色信息层级、黄色主强调、红色播放线/警示、绿色进度/完成。

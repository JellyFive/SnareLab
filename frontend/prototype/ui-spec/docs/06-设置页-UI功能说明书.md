## 页面定位

设置页用于管理声音、播放、数据和响应式偏好。

## 核心功能

| 功能 | 说明 |
|---|---|
| 声音默认值 | 设置节拍器、鼓点、Count-in 默认状态 |
| 默认 BPM | 设置新练习默认 BPM |
| 数据导入导出 | 导入 / 导出本地 JSON |
| 清空缓存 | 清除本地数据，需二次确认 |
| 响应式说明 | 说明当前布局策略 |

## 页面展示内容

### 声音与播放

- 默认节拍器
- 默认鼓点
- Count-in
- 默认 BPM

### 数据

- 导入
- 导出
- 清空缓存

### 响应式策略

说明：

- 大屏完整侧边栏
- 中屏图标侧边栏
- 小屏底部导航
- 视觉风格不变，只改变布局

## 关键交互

| 交互 | 结果 |
|---|---|
| 切换声音默认值 | 更新设置 |
| 修改默认 BPM | 保存默认值 |
| 点击导入 | 选择文件 |
| 点击导出 | 下载 JSON |
| 点击清空缓存 | 弹出二次确认 |

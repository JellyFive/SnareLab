## 页面定位

日志页用于沉淀个人练习过程。

它不使用课堂或教学语境，而是围绕个人练习展开。

## 核心功能

| 功能 | 说明 |
|---|---|
| 练习记录 | 保存每次练习的 BPM、时长、模式 |
| 练习笔记 | 记录想法、灵感、教材片段 |
| 问题记录 | 记录练习中发现的问题 |
| 练习回顾 | 记录一次练习后的总结 |
| 个人目标 | 新建和追踪目标 |
| 关联 | 笔记可关联目标、练习、节奏型 |

## 页面展示内容

### 左侧快速记录

- 新建练习笔记
- 新建个人目标
- 补记练习

### 记录类型筛选

- 全部
- 笔记
- 问题
- 回顾

### 记录列表

每项展示：

- 标题
- 类型
- 关联内容
- 摘要
- 时间

### 新建练习笔记表单

字段：

- 笔记标题
- 类型
- 关联目标
- 标签
- 正文内容

## 关键交互

| 交互 | 结果 |
|---|---|
| 点击新建练习笔记 | 打开笔记表单 |
| 点击新建个人目标 | 打开目标表单 |
| 点击补记练习 | 打开手动练习记录表单 |
| 点击记录项 | 查看详情 |
| 点击保存 | 保存到本地数据库 |
| 关联练习 | 打开练习选择器 |

## 空状态

| 场景 | 显示 |
|---|---|
| 无日志 | 提示“记录第一次练习回顾” |
| 无目标 | 提示“创建个人目标后可关联笔记” |

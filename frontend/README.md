This folder is the frontend workspace.

Current contents:

- `prototype/ui-spec/`: static UI prototype and page-level UI specifications.
- `src/catalog/`: frontend-owned rhythm pattern catalog source files used by the future React application.

The planned frontend stack is React, TypeScript, Vite, Zustand, Dexie, Web Audio API, and VexFlow.
